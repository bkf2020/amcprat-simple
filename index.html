What is this version of AMCPRAT?

This is a simple version of AMCPRAT. It is designed to have less code and a simple interface than the full-featured one. You can view the other version (the full-featured version) here.

How do I use it?

Select one of the links above for your problem set. You can either select AIME, AMC 8, AMC 10, or AMC 12 problems. The difficulty can be Easy, Medium, Hard, or Very Hard.
