What is this version of AMCPRAT?

This is a simple AMC Practice Tool. It is designed to have only HTML and CSS (for the web site). Here is the link to the full-featured AMCPRAT. Python is used to scrape the AMC Problems and to generate the problemsets.

How do I use it?

Select one of the links above for your problemset. You can either select AIME, AMC 8, AMC 10, or AMC 12 problems. The difficulty you choose can be Easy, Medium, Hard, or Very Hard.

Layout inspired by based.cooking, although both websites have different use cases.
